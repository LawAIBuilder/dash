# Alpha Stop Line

The internal alpha is finished enough to use and test when one attorney or paralegal can:

1. Open a matter in the browser.
2. Sync Box and PracticePanther matter metadata.
3. Review and classify documents.
4. Resolve OCR issues in-product.
5. Inspect people and timeline context.
6. Assemble a hearing packet.
7. Export:
   - exhibit list
   - packet preview
   - combined packet output
8. Upload target documents for a second package type.
9. Run one second package type end to end.
10. Review AI-assisted output with citations.
11. Approve final output before anything is written back to Box.

## Required proof for alpha completion

Alpha completion must be demonstrated with:

- a repeatable golden-fixture run
- passing typecheck
- passing tests
- no unresolved P1 cross-case integrity or security blockers
- visible recovery paths for stuck jobs and failed syncs
